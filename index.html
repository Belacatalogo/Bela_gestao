🧪 Desenvolvimento
🧪 Modo Dev — Sandbox de Testes
Ativa um ambiente completamente isolado para testes. Enquanto ativo:
· Firebase desconectado — nenhuma leitura ou escrita no banco real
· localStorage intocado — dados reais nunca são alterados
· Dados fictícios — 4 clientes e 5 vendas de demonstração
· Login automático — não pede senha Google

Ao desativar, o app volta ao normal imediatamente.
📷 Upload de Fotos — ImgBB
🔑 Chave da API ImgBB
Com a chave configurada, você escolhe qualquer foto do celular e o app envia automaticamente para o ImgBB — sem precisar sair do app. Funciona em fotos de clientes e fotos de produtos.

Para obter gratuitamente: acesse imgbb.com → faça login → clique no seu usuário → API → copie a chave.
IA — Análise de Foto
🔑 Chave da API Gemini (Google)
Cole aqui sua chave da API do Google Gemini para ativar a análise automática de fotos de produtos. A chave fica salva só no seu celular e nunca vai para o GitHub ou internet.

Para obter gratuitamente: acesse aistudio.google.com → Get API Key → Create API Key.
Configurações do Catálogo
💰 Mostrar preços no catálogo
Quando ativado, os preços aparecem nos produtos do catálogo para as clientes. Quando desativado, aparece "Consulte" no lugar do preço.

🔄 Sincronizar preços com o catálogo
Envia para o catálogo os preços que você já cadastrou nos produtos aqui no gestão. Assim você não precisa redigitar os preços dos produtos existentes.
📦 Sincronizar produtos com o catálogo
Atualiza o catálogo com os produtos do gestão que têm foto. Produtos sem foto são mantidos no catálogo se já estiverem lá. Só são removidos do catálogo produtos que foram excluídos do gestão.
🧹 Limpar duplicatas (uso único)
Se os produtos estão aparecendo duplicados no carrossel ou no catálogo, este botão remove as cópias indevidas dos produtos fixos (IDs 1–77) que foram criadas por engano. Seus produtos personalizados (IDs ≥ 1000) não são afetados.

Instalar como App
📱 Adicionar à tela inicial
Instale o Bela como um app no seu celular. Funciona offline, abre sem barra do navegador e fica com ícone na tela inicial — igual a um app de verdade.
1
No Brave, toque nos três pontinhos (⋮) no canto superior direito
2
Toque em "Adicionar à tela inicial" ou "Instalar app"
3
Confirme tocando em "Adicionar". O ícone Bela aparecerá na sua tela!
4
Abra sempre pelo ícone — seus dados ficam salvos nele, separados do navegador
